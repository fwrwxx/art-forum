📊 АНАЛІЗ ТА МОДЕЛЮВАННЯ СИСТЕМИ

Актуальність теми

У сучасному цифровому просторі художники змушені використовувати різні платформи для різних потреб. Це створює фрагментацію аудиторії та складність просування власного бренду.

Вирішення проблеми: створення єдиної платформи ArtHub, яка об'єднує форум, магазин та аукціон.

Мета роботи

Розробка адаптивного веб-застосунку для художників ArtHub, що об'єднує функціонал тематичного форуму, онлайн-магазину та аукціону мистецьких робіт.

Бізнес-логіка

КатегоріяПравила
Реєстрація та профілі• Користувач може зареєструватися як "Художник" або "Колекціонер/Покупець"

💻 ТЕХНІЧНА РЕАЛІЗАЦІЯ

Адаптивна сітка (CSS Grid + Media Queries)

.card-grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 24px;
}

@media (max-width: 768px) {
    .card-grid {
        grid-template-columns: repeat(2, 1fr);
    }
}

@media (max-width: 480px) {
    .card-grid {
        grid-template-columns: 1fr;
    }
}

Анімації та Hover-ефекти

.card {
    transition: all 0.3s ease;
}

.card:hover {
    transform: translateY(-10px);
    box-shadow: var(--shadow-gold);
}

.card img {
    transition: transform 0.5s ease;
}

.card:hover img {
    transform: scale(1.05);
}

@keyframes fadeInUp {
    from {
        opacity: 0;
        transform: translateY(30px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

📝 ВИСНОВКИ

У ході виконання лабораторної роботи №1 було:

  • Проаналізовано предметну область та виявлено потребу в єдиній платформі для художників

  • Реалізовано адаптивний інтерфейс з використанням HTML5, CSS3 (Flexbox/Grid) та медіа-запитів
  • Створено бургер-меню з анімацією для мобільних пристроїв
  • Додано інтерактивні елементи та hover-ефекти
  • Організовано роботу з Git та підготовлено документацію

Отримані навички: проєктування інформаційних систем, семантична верстка, адаптивний веб-дизайн, CSS Grid/Flexbox, робота з Git.
