📚 Теоретичні відомості

1. HTTP-сервер

HTTP-сервер — це програмне забезпечення, яке обробляє HTTP-запити від клієнтів (браузерів або інших програм) та повертає відповіді.

Основні функції HTTP-сервера:

  • прийом запитів від клієнтів
  • обробка запитів
  • формування відповіді
  • передача результату клієнту

2. Node.js

Node.js — це середовище виконання JavaScript на стороні сервера.

Переваги Node.js:

  • асинхронність
  • висока продуктивність
  • велика кількість бібліотек
  • можливість створення REST API

3. Express.js

Express.js — це фреймворк для Node.js, який спрощує створення серверних застосунків.

Основні можливості:

  • створення маршрутів
  • обробка HTTP-запитів
  • middleware
  • створення REST API

4. REST API

REST (Representational State Transfer) — це архітектурний стиль для створення веб-сервісів.

Основні принципи REST:

  • використання HTTP-методів
  • клієнт-серверна архітектура
  • відсутність стану (stateless)
  • використання стандартних форматів даних

5. Основні HTTP-методи

GET
отримання даних
POST
створення нового ресурсу
PUT
оновлення ресурсу
DELETE
видалення ресурсу

Створити маршрут POST /students для додавання нового студента.

app.use(express.json());

app.post('/students', (req, res) => {
    const newStudent = {
        id: students.length + 1,
        name: req.body.name,
        group: req.body.group
    };
    students.push(newStudent);
    res.status(201).json(newStudent);
});

📌 Завдання 5

Створити маршрути PUT /students/:id та DELETE /students/:id для оновлення та видалення студента.

// Оновлення студента
app.put('/students/:id', (req, res) => {
    const id = parseInt(req.params.id);
    const student = students.find(s => s.id === id);
    
    if (student) {
        student.name = req.body.name || student.name;
        student.group = req.body.group || student.group;
        res.json(student);
    } else {
        res.status(404).json({ error: 'Student not found' });
    }
});

// Видалення студента
app.delete('/students/:id', (req, res) => {
    const id = parseInt(req.params.id);
    const index = students.findIndex(s => s.id === id);
    
    if (index !== -1) {
        students.splice(index, 1);
        res.status(204).send();
    } else {
        res.status(404).json({ error: 'Student not found' });
    }
});

💻 Повний код сервера (server.js)

const express = require('express');
const app = express();

app.use(express.json());

let students = [
    { id: 1, name: 'Іван Петренко', group: 'ІП-01' },
    { id: 2, name: 'Марія Шевченко', group: 'ІП-01' },
    { id: 3, name: 'Олексій Коваленко', group: 'ІП-02' }
];

// GET /students - отримати всіх студентів
app.get('/students', (req, res) => {
    res.json(students);
});

// GET /students/:id - отримати студента за ID
app.get('/students/:id', (req, res) => {
    const id = parseInt(req.params.id);
    const student = students.find(s => s.id === id);
    
    if (student) {
        res.json(student);
    } else {
        res.status(404).json({ error: 'Student not found' });
    }
});

// POST /students - додати нового студента
app.post('/students', (req, res) => {
    const newStudent = {
        id: students.length + 1,
        name: req.body.name,
        group: req.body.group
    };
    students.push(newStudent);
    res.status(201).json(newStudent);
});

// PUT /students/:id - оновити студента
app.put('/students/:id', (req, res) => {
    const id = parseInt(req.params.id);
    const student = students.find(s => s.id === id);
    
    if (student) {
        student.name = req.body.name || student.name;
        student.group = req.body.group || student.group;
        res.json(student);
    } else {
        res.status(404).json({ error: 'Student not found' });
    }
});

// DELETE /students/:id - видалити студента
app.delete('/students/:id', (req, res) => {
    const id = parseInt(req.params.id);
    const index = students.findIndex(s => s.id === id);
    
    if (index !== -1) {
        students.splice(index, 1);
        res.status(204).send();
    } else {
        res.status(404).json({ error: 'Student not found' });
    }
});

const PORT = 3000;
app.listen(PORT, () => {
    console.log(`Server running on http://localhost:${PORT}`);
});
